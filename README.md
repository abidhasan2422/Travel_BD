# Explore  " Travel Bangladesh" Website

This is a simple and responsive travel website designed using **HTML and CSS**.  
The website highlights popular tourist destinations in Bangladesh such as Cox’s Bazar, Sylhet, Sundarbans, Bandarban, and more.

## Features
- Responsive layout (mobile, tablet, desktop)
- Navigation bar with mobile menu
- Hero section with background image
- Destination cards with hover effects
- “Read More” button on each card
- Clean and simple UI design

## Technologies Used
- HTML5  
- CSS3  
- Git & GitHub Pages (for live hosting)
